# Rust Data Repository

Этот репозиторий предназначен для автоматической выгрузки и хранения данных, полученных из сервера или клиента игры Rust.

## Назначение

* Автоматический сбор данных.
* Хранение актуальной информации.
* Использование данных для анализа, разработки инструментов и исследований.

## Источник данных

Данные могут автоматически выгружаться:

* с игрового сервера Rust;
* с клиента Rust.

Структура и состав данных могут изменяться по мере развития проекта.

## Обновление

Содержимое репозитория обновляется автоматически короме клиента они ручные.
